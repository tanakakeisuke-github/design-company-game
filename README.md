# デザイン会社発展国 🎨

> デザイン会社の経営シミュレーションゲーム  
> 2畳のコワーキングスペースから、世界的スタジオを目指せ。

[![Play](https://img.shields.io/badge/▶%20Play%20Now-GitHub%20Pages-4f46e5?style=for-the-badge)](https://YOUR_USERNAME.github.io/design-company-game/)

## 🎮 遊び方

1. **案件タブ** で案件を選んで受注
2. **制作タブ** で社員をアサイン
3. 自動で週が進む（AUTOボタンでON/OFF）
4. 収益で社員採用・技術習得・オフィス拡張
5. 7種類のエンディングを目指す

### ⌨️ キーボードショートカット

| キー | 操作 |
|------|------|
| `Space` | 次の週へ |
| `A` | 自動進行 ON/OFF |
| `1〜7` | タブ切替 |
| `Ctrl+S` | セーブ |

## 📊 ゲームシステム

### パラメータ
| アイコン | 名前 | 説明 |
|----------|------|------|
| 💰 | 資金 | 月次コスト（給与+家賃）が払えないとゲームオーバー |
| ⭐ | 評判 | 高いほど高単価案件が来る（0〜100） |
| 🔥 | 炎上 | 90%超えで炎上エンド |
| 📢 | SNS | フォロワー数。成功で伸びる |
| 😈 | ブラック度 | 高いと社員が辞める |
| 🧠 | メンタル | 社員ごとの疲労度。低いと仕事が遅くなる |

### フェーズ
| フェーズ | 名称 | 解放条件 |
|----------|------|----------|
| 0 | 個人事務所 | スタート |
| 1 | デザイン事務所 | 収益150万 + 社員2名 |
| 2 | デザインスタジオ | 収益800万 + 社員5名 |
| 3 | デザイン会社 | 収益5000万 + 社員8名 |

### 案件タイプ
ロゴ・名刺・LP・バナー・アプリUI・Webサイト・ブランディング・AIデザイン

### クライアントタイプ
神クライアント / 普通 / スタートアップ / 代理店 / プレッシャー系 / 官公庁 / ベンチャー / インディー
