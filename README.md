> フリーランスから始めて、デザイン会社を世界企業へと育てるシミュレーションゲーム

**[▶ Play Now](https://your-username.github.io/design-company-game/)**

## 概要

「デザイン会社発展国」は、UIデザイナーのリアルな日常を題材にした経営シミュレーションゲームです。  
ブラウザだけで動く単一HTMLファイルで、インストール不要。

### ゲームの流れ

```
個人事務所 → デザイン事務所 → スタジオ → 会社 → プロダクト企業 → プラットフォーム → テックジャイアント
```

フリーランスの一人から始まり、社員を雇い、オフィスを拡大し、最終的にはグローバルな技術企業を目指します。

---

## 特徴

### 🏢 経営システム
- **7フェーズ**のオフィス進化（コワーキングスペース → グローバルHQ）
- **53種類**の案件タイプ（ロゴ・アプリUI・XRプラットフォームまで）
- 社員の採用・育成・メンタル管理
- 技術ツリーの解放と研究開発
- DAU / ARR などのプロダクトメトリクス
- VC調達・IPO・M&Aイベント

### 🎨 アイソメトリックオフィスビュー
- リアルタイムで動く社員キャラクター
- 吹き出しでリアルな「デザイナーの呟き」
- フェーズに応じてオフィスが自動進化
- ズームイン・アウト、高さリサイズ対応

### 🤖 AIモード
- AIエージェントが受注・採用・案件配分を全自動処理
- Anthropic APIキーを設定するとAIが**実際のSVGデザインを生成**してポートフォリオに蓄積

### 📨 リアルオーダーシステム
- `index.html?order` でクライアント向けオーダーフォームが開く
- **受注コード**（Base64エンコード）でリアルな依頼がゲーム内案件として着弾
- ゲームとリアルが一体化した体験

### 🖼️ 作品集（ポートフォリオ）
- 完了案件が自動でポートフォリオに蓄積
- APIキー設定で **Claude Haiku が自動でSVGデザインを生成**
- ビューワーでSVGを拡大表示・ダウンロード可能
- Midjourney プロンプトも自動生成

## 使い方

### シングルファイルで動作
```
index.html をブラウザで開くだけ
```

### GitHub Pages での公開
1. このリポジトリを fork またはクローン
2. Settings → Pages → Source を `main` ブランチの `/ (root)` に設定
3. `https://your-username.github.io/design-company-game/` でプレイ可能

### リアルオーダーページの共有
```
https://your-username.github.io/design-company-game/?order
```
クライアントにこのURLを共有すると、ゲーム内にオーダーが届きます。

### AI生成デザインの有効化
1. 作品集タブを開く
2. 上部の「🤖 AI生成」バーに Anthropic API キー（`sk-ant-...`）を入力
3. 案件完了のたびに Claude Haiku が自動でSVGデザインを生成
