## 遊び方

1. `design_company_game.html` をブラウザで開く
2. 会社名を入力してゲームスタート
3. 「▶ 次の週へ」で時間を進める
4. 案件を受注 → アサイン → 完了を繰り返す
5. 収益で採用・技術習得・オフィス拡張

### 🤖 AIモード
全経営をAIに委任するオートプレイ機能。速度は3段階で調整可能。

### ⚙️ 管理パネル（DEVボタン）
ゲームパラメータをリアルタイムに調整できる開発者向け機能。
