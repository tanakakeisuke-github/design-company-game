# 🎨 Design Company Game

デザイン会社の経営シミュレーションゲーム。個人事務所から始め、テックジャイアントを目指す。

**→ [Play Now](https://your-username.github.io/design-company-game/)**

---

## ゲーム概要

あなたはフリーランスデザイナー。受注・採用・技術習得を積み重ねて、会社を成長させよう。

### フェーズ
| Phase | 形態 | 社員上限 |
|---|---|---|
| 0 | 個人事務所 | 4名 |
| 1 | デザイン事務所 | 10名 |
| 2 | デザインスタジオ | 16名 |
| 3 | デザイン会社 | 22名 |
| 4 | 🚀 プロダクト企業 | 200名 |
| 5 | 🌐 プラットフォーム企業 | 1,000名 |
| 6 | ⚡ テックジャイアント | 5,000名 |

### 主な要素
- 📋 **案件受注** — ロゴ・バナー・Webサービス・AIプロダクトなど多種多様
- 👥 **社員採用** — スキル・メンタル・リスクを見極めながら採用
- 🤖 **AIエージェント** — 経営をAIに全委任するモード搭載
- 🏢 **オフィス移転** — コワーキングスペース→渋谷→グローバルHQ
- 📱 **DAU/ARR** — フェーズ4以降は自社プロダクトの成長指標が登場
- 🎲 **イベント** — デザイン賞・炎上・VC投資・M&Aなど30種以上

### エンディング
- 🌍 世界的デザインスタジオ
- 🚀 プロダクト企業への転身
- ✨ 神スタジオ（Best Ending）
- 💀 ブラック崩壊エンド
- など全8種

## 技術情報

- **Single HTML file** — サーバー不要、ブラウザで即プレイ
- **セーブ機能** — localStorage に自動保存
- 外部依存なし（Google Fonts のみ）

## GitHub Pages でのデプロイ

1. このリポジトリをフォーク or クローン
2. Settings → Pages → Source: `main` ブランチ / `/ (root)`
3. `https://your-username.github.io/design-company-game/` でプレイ可能

## ローカルでのプレイ

```bash
git clone https://github.com/your-username/design-company-game.git
cd design-company-game
open index.html  # macOS
# または index.html をブラウザにドラッグ＆ドロップ
```
