## 🎮 ゲーム概要

### 基本の流れ
1. **案件タブ** で仕事を受注する
2. **制作タブ** で社員をアサインする
3. **「次の週へ」** で時間を進める
4. 完了 → 報酬を受け取り、評判が上がる
5. 収益で**社員採用・技術習得・オフィス拡張**

### 🤖 AIモード
全経営判断（受注・採用・アサイン・技術・移転）を自動化。  
スピード ×1 / ×2 / ×4 で進行速度を調整可能。

## 🌿 ウェルビーイングシステム

ブラック企業になることなく成長するための制度システム。

- **WBスコア (0〜100)** で組織の健康度を管理
- 20種のポリシー（フレックス・4日制・育児支援・DEIなど）を採用
- WBが高いほど社員のパフォーマンスが上がり、ブラック度が下がる
- **スター社員**（stats平均80+・メンタル80+）が自動検出される

## 🎨 オフィスビジュアル

アイソメトリックキャンバスでリアルタイム描画。

- レベルに応じた**テーマ切り替え**（京都和風 / モダン / ハイテクダーク）
- 社員キャラクターがリアルタイムで歩き・作業
- 提灯・障子・畳など京都要素を細かく表現
- ズーム・ドラッグリサイズ対応

## 🛠️ 開発・カスタマイズ

HTMLを直接編集するだけでカスタマイズ可能。主要定数：

| 定数 | 内容 |
|------|------|
| `PROJ_TYPES` | 案件種類の定義 |
| `HIRE_POOL` | 採用候補スタッフ |
| `TECH_TREE` | テクノロジーツリー |
| `POLICIES` | 制度・ポリシー |
| `OFFICES` | オフィスレベル |
| `ENDINGS` | エンディング条件 |
| `EVENTS` | ランダムイベント |
